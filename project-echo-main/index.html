Project Echo

A developer profile app for David Timothy, showcasing projects, achievements, and a knowledge reflection built with modern web tools.

Data is loaded from JSON files via the Fetch API and rendered into Bootstrap cards.

Bootstrap 5 jQuery Fetch API JSON Data zero-md

User Story

As a college student building a tech career,

I want one place to quickly show my projects, achievements, and what I’ve learned,

So that classmates, professors, and recruiters can understand my skills and growth without digging through multiple links.
